### delicate开源主题 （支持响应式）

### 配置：
 ```xml
配置文件：src/main/resources/application.yml

上传/轮播：thumbnail_post_size: 1920x1080（最佳） banner

文章/配图：storage_max_width: 800 （最佳） markdown0

导航缩图：thumbnail_post_size: 200x200 （图标）
 ```
